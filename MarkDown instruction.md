# Инструкция для работы с MarkDown

## Выделение текста 

Чтобы выделить текс курсивом необходимо обрамить его звездочками (*). 
Например, *вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочками (**)..
Например **Вот так**
