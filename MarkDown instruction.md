## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочками (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Первый Элемент
2. Второй Элемент
